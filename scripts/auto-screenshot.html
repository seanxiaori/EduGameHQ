🎯 EduGameHQ Product Hunt 截图工具

这个工具帮你快速预览和截图EduGameHQ的关键页面，用于Product Hunt提交。

📋 使用说明：

  1. 确保你的开发服务器在 http://localhost:3000 运行
  2. 点击"加载预览"按钮查看页面
  3. 使用浏览器的截图功能（F12 → 响应式模式 → 设置1270x760 → 截图）
  4. 保存截图到 public/product-hunt-assets/ 目录

1. 首页截图 (Homepage)

目标尺寸: 1270x760px | 文件名: homepage-screenshot.png

截图要点: 展示完整的首页布局、导航栏、游戏分类卡片、特色游戏区域

2. 游戏分类页面截图 (Category Page)

目标尺寸: 1270x760px | 文件名: category-page-screenshot.png

截图要点: 展示游戏列表、筛选功能、游戏卡片设计、分类导航

3. 游戏详情页面截图 (Game Detail)

目标尺寸: 1270x760px | 文件名: game-detail-screenshot.png

截图要点: 展示游戏界面、游戏信息、描述文字、相关推荐

🎯 快速截图步骤

  1. 打开新标签页: 点击"在新标签页打开"按钮
  2. 进入响应式模式: 按 F12 → Ctrl+Shift+M
  3. 设置尺寸: 在顶部输入 1270 x 760
  4. 截图: 点击右上角的相机图标
  5. 保存: 下载并重命名文件
💡 提示: 如果页面加载有问题，请确保开发服务器正在运行 (npm run dev)
