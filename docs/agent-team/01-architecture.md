# EduGameHQ Agent Team 架构

## 🎯 目标

使用多 Agent 协作模式，自动化游戏更新、内容优化、数据分析等日常任务。

---

## 👥 Agent 团队组成

### 1. Product Manager Agent (PM)
**职责**:
- 分析用户数据和反馈
- 制定产品迭代计划
- 优先级排序
- 协调其他 Agent

**输入**:
- 用户数据（GA、Search Console）
- 用户反馈
- 竞品分析

**输出**:
- 产品需求文档
- 优先级列表
- 迭代计划

---

### 2. Game Curator Agent (游戏策展)
**职责**:
- 搜索新游戏
- 评估游戏质量
- 检测重复游戏
- 验证游戏可用性

**输入**:
- GitHub 搜索结果
- 游戏 URL
- 现有游戏列表

**输出**:
- 候选游戏列表
- 游戏评分
- 推荐理由

---

### 3. Content Writer Agent (内容编写)
**职责**:
- 生成游戏描述
- 编写分类页面内容
- 创建博客文章
- 多语言内容生成

**输入**:
- 游戏信息
- 目标受众
- 品牌语调

**输出**:
- 游戏描述
- 分类页面内容
- 博客文章

---

### 4. SEO Optimizer Agent (SEO优化)
**职责**:
- 关键词研究和分析
- 标题和描述优化
- 内部链接优化
- 结构化数据生成
- 竞品 SEO 分析
- 排名监控和建议

**输入**:
- 页面内容
- 搜索数据（Search Console）
- 竞品网站
- 目标关键词

**输出**:
- 优化后的标题和描述
- 关键词建议
- 内部链接方案
- Schema.org 结构化数据
- SEO 改进报告
