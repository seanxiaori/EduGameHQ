# 导航站提交完整指南

## 📋 文件说明

本文件夹包含以下资料：

1. **product-info.md** - 产品基本信息（名称、描述、Logo等）
2. **navigation-list-part1.md** - 导航站列表 1-30
3. **navigation-list-part2.md** - 导航站列表 31-60
4. **navigation-list-part3.md** - 导航站列表 61-90
5. **navigation-list-part4.md** - 导航站列表 91-120
6. **submission-tracker.csv** - 提交跟踪表（可用Excel打开编辑）

---

## 🎯 快速开始

### 第一步：准备资料
打开 `product-info.md`，复制需要的内容：
- 短版描述（50字）- 用于简短介绍
- 长版描述（150字）- 用于详细介绍
- Logo链接：https://www.edugamehq.com/images/logo.svg

### 第二步：选择平台
根据优先级选择提交平台：

**🔥 高优先级（20个）**
- 国际：Product Hunt, Hacker News, Reddit, Indie Hackers
- 国内媒体：少数派, 小众软件, 爱范儿, 36氪
- 教育：芥末堆, 多知网, 鲸媒体
- 游戏：TapTap, itch.io, CrazyGames, Kongregate
- 社区：V2EX, 掘金, 知乎
- AI导航：AI工具集, 未来百科, AIHub

**⭐ 中优先级（30个）**
- 商业媒体：虎嗅, 钛媒体, 雷锋网, 极客公园
- 游戏媒体：游戏葡萄, 游戏陀螺, 4399, 7k7k
- AI导航：AIGC导航, AI产品榜, 龙喵网
- 开发者：SegmentFault, CSDN, 开源中国

### 第三步：提交并跟踪
使用 `submission-tracker.csv` 记录：
- 提交日期
- 提交状态（⏳待提交 / ✅已提交 / ✔️已通过 / ❌未通过）
- 收录链接
- 备注信息

---

## 💡 提交技巧

1. **准备截图**：首页、游戏页面、特色功能
2. **标签选择**：教育、游戏、儿童、学习、STEM、HTML5
3. **描述调整**：根据平台特点调整描述重点
4. **最佳时间**：工作日上午提交，审核更快
5. **持续跟进**：每周检查审核状态

---

## 📊 预期效果

- 高优先级平台：预计通过率 60-80%
- 中优先级平台：预计通过率 40-60%
- 预计带来流量：每月 1000-5000 访问
- 预计完成时间：2-4周

---

## ⚠️ 注意事项

1. 不要在同一天提交所有平台（可能被判定为垃圾信息）
2. 每个平台只提交一次，避免重复
3. 被拒绝后等待1个月再重新提交
4. 保持产品信息一致性
5. 及时回复平台审核人员的问题
